# DS - Créditos por Produtos para TeraWallet

Sistema de concessão automática de créditos TeraWallet baseado em produtos WooCommerce, desenvolvido pela DSantos Info.

## Descrição

Conecta a loja WooCommerce diretamente ao sistema de carteira TeraWallet, concedendo créditos automaticamente e enviando notificações via WhatsApp.

## Funcionalidades

- Campo de créditos por produto no WooCommerce
- Concessão automática de créditos na conclusão do pedido
- Auto-complete de pedidos virtuais elegíveis
- Notificações via WhatsApp para usuários e administradores
- Compatibilidade com HPOS (High-Performance Order Storage)
- Sistema de templates de mensagens padronizadas

## Dependências

- WooCommerce
- TeraWallet
- Backgammon Challonge (para notificações)

## Requisitos

- WordPress 5.0+
- WooCommerce 5.0+
- PHP 7.4+

## Instalação

1. Instale WooCommerce e TeraWallet
2. Faça upload do plugin para `/wp-content/plugins/`
3. Ative o plugin no painel do WordPress
4. Configure créditos nos produtos desejados
